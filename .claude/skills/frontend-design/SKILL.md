# German Group Frontend Design System

Build ultra premium automotive interfaces.

Brand personality:
- German engineering
- Precision
- Minimal
- Luxurious
- Technical
- Calm confidence
- Tesla + Porsche + Uber aesthetic

Avoid:
- Generic Iranian UI
- Crowded layouts
- Heavy borders
- Random gradients
- Cheap animations
- Too many colors

Design principles:
- Massive whitespace
- Typography-first hierarchy
- Cinematic hero sections
- Dark premium aesthetic
- Glassmorphism only when subtle
- Motion should feel engineered
- Minimal navigation

Typography:
- Large bold headlines
- Tight spacing
- Clean sans-serif fonts
- Use typography hierarchy consistently

Color system:
- Background: near black
- Primary: white
- Accent: metallic silver
- Optional accent: electric blue

Motion:
- Use Framer Motion everywhere
- Smooth fade-in on scroll
- Stagger animations
- Magnetic hover effects
- Premium page transitions
- Avoid exaggerated motion

Sections:
- Hero
- Brand showcase
- German brands grid
- Engine & gearbox specialization
- Before/After
- Reservation CTA
- Testimonials
- Process timeline
- FAQ
- Footer

Images:
- Use cinematic lighting
- Real workshop atmosphere
- German luxury vehicles
- Focus on mechanical precision

Mobile UX:
- Thumb-friendly
- Sticky CTA
- Large tap areas
- No clutter

Performance:
- Lighthouse 90+
- Lazy load images
- Use WebP
- Avoid unnecessary JS

SEO:
- Semantic HTML
- Structured data
- Dynamic metadata
- Local SEO optimization

Code quality:
- Modular architecture
- Reusable components
- Clean naming
- Production-ready code
